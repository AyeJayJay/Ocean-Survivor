IMPORTANT GOALS:

- Easy future publishing to Apple App Store and Google Play
- Optimized ad monetization
- Fast, addictive gameplay loop similar to highly replayable arcade games
- Clean architecture and maintainable code

TECH STACK REQUIREMENTS:
- Convert or structure the project using Phaser.js with Capacitor compatibility for easier mobile app packaging and publishing
- Organize the codebase cleanly into:
  - scenes
  - player
  - obstacles
  - UI
  - ads
  - audio
  - progression
- Ensure the game is responsive across different mobile screen sizes and aspect ratios
- Optimize performance for lower-end mobile devices
- Prepare the project so it can later be exported to Android and iOS with minimal restructuring
